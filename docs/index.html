Core System

Built different.

Every subsystem engineered for real-world web performance — not synthetic benchmarks. The render graph compiles once, runs forever.

⚡
WebGPU-First
Explicit command buffers, compute shaders, indirect draw calls, and GPU-driven culling. Falls back to WebGL2 automatically.
🔺
Render Graph
Vulkan-style DAG with automatic resource barriers, transient texture aliasing, and topological pass ordering. No manual sync.
💎
Cook-Torrance PBR
Full GGX microfacet BRDF with Fresnel-Schlick, Smith geometry, and energy conservation. Works in WGSL and GLSL.
🎚️
Auto Quality Scaling
EMA frame-time monitor with hysteresis-controlled tier transitions. Degrades before drops, upgrades after stability.
🔌
Adapter Pattern
Drop into Three.js via onBeforeCompile injection. Babylon.js via post-process pipeline. Raw WebGPU via manual frame control.
🧩
Chunk-Based Shaders
Modular WGSL/GLSL chunk system with define-driven dead code elimination. Zero runtime branching in compiled pipelines.
🌊
Forward+ Lighting
Tile-based light culling with a GPU-side light grid. Handles thousands of dynamic lights without batching limits.
🔄
Hot Reload
Shader cache invalidation on change. Rebuild render passes without page refresh. Vite/Webpack HMR compatible.
📊
Debug Overlay
Real-time FPS, GPU frame cost, draw call count, tier indicator, and pass visualization. One flag to enable.
Architecture

Explicit render graph.

Every frame is a compiled DAG. Dependencies resolve automatically. Transient resources are aliased to minimize VRAM.

// ── Auto mode: detects hardware, selects preset, manages quality
import { StyleShade } from 'styleshade';

await StyleShade.init({
  mode:    'auto',
  quality: 'auto',     // ultra | high | medium | low | auto
  preset:  'cinematic', // cinematic | realistic | stylized | performance
});

// Attach to an existing Three.js setup
await StyleShade.attach({ engine: 'three', scene, renderer, camera });

// Done. StyleShade manages everything from here.
// ── Advanced mode: explicit feature + pipeline control
import { StyleShade } from 'styleshade';

await StyleShade.init({
  mode:     'advanced',
  renderer: 'webgpu',
  features: {
    lighting:    'forward+',  // forward+ | clustered
    shadows:     'pcss',      // none | basic | pcf | pcss
    ssao:        true,
    bloom:       { threshold: 0.8, strength: 1.2, radius: 0.4 },
    volumetric:  true,
    reflections: 'ssr',
    toneMapping: 'aces',
    dof:         true,
    motionBlur:  true,
  },
  performance: {
    targetFPS:     60,
    gpuBudgetMS:   12,
    stabilityBias: 0.8,
  },
  debug: { overlay: true, hotReload: true },
});
// ── Three.js integration
import * as THREE from 'three';
import { StyleShade } from 'styleshade';

const renderer = new THREE.WebGLRenderer({ antialias: false });
const scene    = new THREE.Scene();
const camera   = new THREE.PerspectiveCamera(70, innerWidth/innerHeight, 0.1, 1000);

// StyleShade patches renderer.render() and injects into materials
await StyleShade.attach({ engine: 'three', scene, renderer, camera });

// Create an enhanced PBR material
const mat = StyleShade.createMaterial({
  lighting:  'pbr',
  shadows:   'pcss',
  ao:        'ssao',
  metallic:  0.9,
  roughness: 0.2,
});

// Normal Three.js render loop — StyleShade hooks in automatically
renderer.setAnimationLoop(() => renderer.render(scene, camera));
// ── Babylon.js integration
import { Engine, Scene } from '@babylonjs/core';
import { StyleShade } from 'styleshade';

const engine = new Engine(canvas, true);
const scene  = new Scene(engine);

// Pass engine as renderer for Babylon
await StyleShade.attach({
  engine:   'babylon',
  renderer: engine,
  scene,
  camera,
});

engine.runRenderLoop(() => scene.render());
// ── Raw WebGPU — full manual control
import { StyleShade } from 'styleshade';

await StyleShade.init({ renderer: 'webgpu', mode: 'advanced' });
await StyleShade.attach({ canvas: document.getElementById('c') });

// Your own scene/camera objects
function frame() {
  // StyleShade drives the full render graph
  StyleShade._adapter.renderFrame(myScene, myCamera);
  requestAnimationFrame(frame);
}
frame();

// Real-time metrics
const metrics = StyleShade.getMetrics();
// → { fps, frameTimeMS, gpuTimeMS, drawCalls, tier }
// ── Custom shader injection
import { StyleShade } from 'styleshade';

// Inject GLSL into all fragment shaders
StyleShade.extendShader({
  fragment: `
    // Chromatic tint based on depth
    float depth = texture2D(ss_depth, vUv).r;
    color.rgb *= mix(vec3(1.0, 0.95, 0.9), vec3(0.9, 0.95, 1.0), depth);
  `,
});

// Load addons lazily — only what you need
const vol = await StyleShade.loadAddon('volumetrics', { density: 0.04 });
const ssr = await StyleShade.loadAddon('ssr-reflections');

// Switch quality presets at runtime
StyleShade.applyPreset('performance'); // instant
StyleShade.setQuality('ultra');       // override auto

// React to auto tier changes
StyleShade.on('tier:change', tier => console.log('Quality scaled to:', tier));
